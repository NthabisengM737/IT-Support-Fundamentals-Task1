## Problem Statement
the purpose of this practical task is to demonstrate the ability to document system specifications, verify network connectivity, simulate basic issue (internet disconnection), and resolve it. This exercise highlights fundamental trobleshooting and documentation skills required for managing operating systems in real or virtual environments.

## 3. Simulated Issue

## 4. Resolution
